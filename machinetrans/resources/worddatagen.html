English Sentence Generation

This is the main page for the English sentence generator. Here you can select a given type of sentence to generate by click on the list and pressing the "Accept" button. Once you do this, a list of sentence varieties for that type of sentence will be displayed. Here you click an item, press the "Accept" button and the given sentence variety will be displayed with random English data from the database. At any time you can press the "Back" button and get the main menu. For more information about the whys and whererfores of this program read the descriptive document included with the program in the file named "GrammaticalMachineTranslation." Eventually this will be the English producing end of the machine translation program.
