SQL Data

The SQL data page allows you to review, edit or cancel the data in the database for the previously selected type of word. To edit the data: Select a datum or group of data and press "Edit" at the bottom an edit dialog will appear. To delete data select the data and then press "Delete" button. To change the sorting order press the "Sort" button and the sorting GUI will appear. To find data press the "Find" button and a dialog will appear to enter the text to find, which can be either in Russian or English. Pressing the "Back" button will bring up the main menu. Any changes you make in this page will be reflected in the database.
